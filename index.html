A whole new set of features

Symbolic computation

Symbolic computation has been removed from Epsilon in version 11.2. It has been reintroduced in Omega.

Periodic table

Inspired by the periodic table app of the TI83PCE, Omega's periodic table is clean and simple to use.

Support of KhiCAS and external apps

Delta, another custom firmware which supports external apps, is fully integrated into Omega. This allows you to run KhiCAS or Nofrendo on your Numworks.
Due to hardware limitations, this is only possible on N0110.

RPN

Omega supports inputting expression using Reverse Polish Notation.
